# 云平台快速入门指南

## 为什么选择云平台？

✅ **性价比高** - RTX 3090 只要 ¥1-2/小时
✅ **速度快** - 国内网络，速度快且稳定
✅ **即开即用** - 预装 PyTorch + CUDA 环境
✅ **灵活租赁** - 按小时计费，用完即停

## 推荐平台

### 1. AutoDL
- 🔗 https://www.autodl.com/
- 💰 RTX 3090: ~¥1.5/小时
- ✨ 预装 PyTorch 环境
- 📱 提供 Web 终端 + SSH

### 2. 智星云
- 🔗 https://www.gpuhub.com/
- 💰 RTX 3090: ~¥1.8/小时
- ✨ 提供桌面环境
- 📱 支持多种深度学习镜像

## 快速开始（5 分钟）

### 第一步：注册平台
1. 访问 AutoDL 或 智星云
2. 注册账号并实名认证
3. 充值（建议先充 ¥50 试试）

### 第二步：租用 GPU
1. 选择"社区镜像" → 搜索 "PyTorch"
2. 推荐镜像：`PyTorch 2.3.0 + Python 3.10 + CUDA 11.8`
3. 选择 GPU：
   - **学习推荐**：RTX 3090 (24GB显存)
   - **高性能**：RTX 4090 (24GB显存)
   - **大模型**：A100 (40GB/80GB)
4. 点击"租用"（按小时计费）

### 第三步：上传代码

**方法 1: Git Clone（推荐）**
```bash
# 在平台的 Web 终端中运行：
git clone <你的仓库地址>
cd cuda_operators
```

**方法 2: 直接上传**
1. 打开平台的 Web 界面
2. 点击 "Upload" 上传项目 zip 文件
3. 在 Terminal 中解压

### 第四步：启动环境
```bash
# 运行自动配置脚本
bash cloud_quickstart.sh
```

### 第五步：开始学习
```bash
# 运行单个算子测试
python tests/test_layernorm.py

# 运行所有测试
python -m pytest tests/ -v

# 运行性能测试
python benchmark/benchmark.py
```

## 使用技巧

### 💡 省钱技巧
1. **用完即停** - 不用时关机，停止计费
2. **选择合适GPU** - 学习用 3090 即可
3. **使用快照** - 保存环境，下次快速恢复

### 💡 性能优化
```bash
# 监控 GPU 使用情况
watch -n 1 nvidia-smi

# 查看详细指标
nvidia-smi --query-gpu=utilization.gpu,utilization.memory,memory.total,memory.used --format=csv
```

### 💡 数据持久化
- **AutoDL**: 数据保存在 `/root/data/` 目录
- **智星云**: 使用云盘存储
- **建议**: 定期下载重要代码到本地

## 故障排除

### ❌ 编译错误
```bash
# 检查环境
nvidia-smi
python -c "import torch; print(torch.__version__)"

# 重新安装依赖
pip install --upgrade pybind11 pytest
```

### ❌ CUDA 版本不匹配
```bash
# 查看当前 CUDA 版本
nvcc --version

# 重装匹配的 PyTorch
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```
